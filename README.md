##  Project Overview

This project focuses on:
- Predicting **stock price** for upcoming days  
- Generating **Buy / Sell / Hold** decisions  
- Using **KNN Regressor** for numerical prediction  
- Using **KNN Classifier** for trading signals  
- Applying technical indicators like MA, EMA, RSI, MACD etc.
